# AI Study Tracker

A full-stack AI Study Tracker application built using Node.js and Express.

## Features
- Add subjects
- Persist data using JSON storage
- Simple and clean UI
- Backend API with Express

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Storage: JSON file
- Version Control: Git & GitHub

## Project Structure
ai-study-tracker/
│
├── server/
